# Installation

## pip

```bash
pip install mcpplay
```

## uv

```bash
uv add mcpplay
```

## pipx

```bash
pipx install mcpplay
```

## From source

```bash
git clone https://github.com/gauthierpiarrette/mcpplay
cd mcpplay
uv sync --dev
```

## Verify

```bash
mcpplay --version
```

!!! info "Requirements"
    Python 3.11+ and a modern web browser. No Node.js or Docker needed.
